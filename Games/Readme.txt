instruções:

1) baixe um pygame para windows com extensaõ .exe ou .bat

2) crie um atalho do aplicativo .exe e salve em qualquer lugar da pasta Retropython-master.

3) Abra o banco de dados no sqlite studio, e apague todos os games

4) crie as linhas preenchendo as seguintes informações:  

id = id do jogo
nome = nome do jogo
imagem = arquivo png,jpg da capa do jogo
descrição = descrição do game
source_windows = caminho relativo do atalho do jogo, exemplo Games/Run!/main.exe-atalho.
source_linux= ainda não funciona deixe vazio - > ''
